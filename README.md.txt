# My Pygame Project

このプロジェクトは、Pygameを使用して作成したゲームです。

## インストール方法

1. リポジトリをクローン：
    ```bash
    git clone https://github.com/Kakegawa10/sotsu.git
    ```
2. 必要なライブラリをインストール：
    ```bash
    cd python
    pip install -r requirements.txt
   

3. ゲームを実行：
    ```bash
    python main.py
    ```
